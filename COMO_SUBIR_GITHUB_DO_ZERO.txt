PASSO A PASSO — SUBIR O PROJETO DO ZERO NO GITHUB

1) Criar repositório novo no GitHub
- Entrar em github.com
- Clicar no + no canto superior direito
- New repository
- Nome sugerido: tech-challenge-fase-4-obesidade-streamlit
- Marcar Public
- Marcar Add a README file: DESMARCADO, pois este pacote já tem README.md
- Clicar em Create repository

2) Subir os arquivos
- No repositório vazio, clicar em uploading an existing file ou Add file > Upload files
- Arrastar TODOS os arquivos desta pasta para o GitHub:
  app.py
  requirements.txt
  README.md
  df_clean.csv
  Obesity.csv
  random.joblib
  pipeline_treinamento.py
  entrega_links.txt
- Clicar em Commit changes

3) Conferir estrutura final
O repositório precisa ficar assim:
app.py
requirements.txt
README.md
df_clean.csv
Obesity.csv
random.joblib
pipeline_treinamento.py
entrega_links.txt

4) Streamlit Cloud
- Entrar em streamlit.io/cloud
- New app ou Create app
- Escolher o repositório novo
- Branch: main
- Main file path: app.py
- Deploy

5) Depois de publicar
- Copiar o link do app Streamlit
- Copiar o link do GitHub
- Atualizar o arquivo entrega_links.txt com os links finais
